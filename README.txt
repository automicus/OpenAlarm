OpenAlarm

DESCRIPTION:
OpenAlarm is a Python program made for Linux that will trigger HTTP API calls 
when the alarm "sounds". The alarm is controlled primarily through its own 
HTTP API and is designed specifically to be a flexible alarm clock engine for 
use in conjunction with OpenRemote? and similar remote applications.
